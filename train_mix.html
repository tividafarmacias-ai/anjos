📊 Processo de MIX de Produtos

Guia completo para análise, construção e entrega do MIX ideal da loja.

1. Dados Necessários

  • Estoque da loja em XLS com EAN completo
  • Relatório de giro de produtos (BI)
📷 INSERIR IMAGEM DO POWER BI / ENTRADA NA BÚSSOLA

2. Extração no Power BI

  • Acessar o Power BI
  • Entrar em Bússola UTC
  • Filtrar pelo nome da loja (PDV)
  • Selecionar categoria
  • Tirar print em "Reais"
📷 PRINT DO FILTRO + CATEGORIA

3. Exportação de Dados

  • Entrar em UTC x Loja
  • Verificar filtros (apenas loja)
  • Clicar em "..."
  • Exportar → Dados com layout atual
📷 PRINT EXPORTAÇÃO POWER BI

4. Construção das Bases

  • Base do mercado (BI)
  • Estoque da loja (TRIER)
  • Média de vendas da rede
⚠️ Configuração do estoque:
  • Saldo maior que zero
  • Com custo
  • EAN completo
  • Descrição completa

5. Tratamento das Planilhas

  • Desfazer mesclagem
  • Padronizar colunas
  • Copiar dados para planilha MIX
  • Colar:
    • Aba estoque
    • Aba base
    • Aba média rede

6. Ajustes Técnicos (Excel)

  • Converter EAN para número
  • Remover casas decimais
  • Aplicar comandos:
    • Ctrl + Shift + M
    • Ctrl + Shift + I
    • Ctrl + Shift + O
    • Ctrl + Shift + L

7. Validação do MIX

  • Atualizar colunas de estoque e venda
  • Transformar fórmulas em valores
  • Testar aba de pedidos

8. Análise Estratégica

  • Remover fabricantes sem relevância
  • Destacar oportunidades (ex: Cristália, Galderma)
  • Comparar com mercado

9. Entrega Final

  • Criar arquivo final com:
    • Aba Análise
    • Comparativo (com print)
    • Ação (plano para loja)
  • Apresentar ao associado
📷 PRINT FINAL PARA APRESENTAÇÃO
